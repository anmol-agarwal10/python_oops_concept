# python_oops_concept
It has all concept of python OOPS, for learning and recognising. 
# Python OOP (Object-Oriented Programming)

## 1. Python OOP Concepts

### 1. Class and Object
- Class: Blueprint for creating objects
- Object: Instance of a class

### 2. Instance Variables and Methods
- Instance variables: Unique to each object
- Instance methods: Operate on instance data

### 3. Class Variables and Class Methods
- Class variables: Shared across all instances
- Class methods: Use `@classmethod` decorator, operate on class level

### 4. Static Methods
- Defined using `@staticmethod`
- Do not depend on class or instance variables

### 5. Encapsulation
- Binding data and methods together
- Restricting direct access using private/protected members

### 6. Polymorphism (Overloading / Overriding)
- Ability to use same interface for different data types
- Method overloading (simulated in Python)
- Method overriding (runtime polymorphism)

### 7. Abstraction
- Hiding implementation details
- Achieved using abstract classes (`ABC` module)

### 8. Inheritance
- Single inheritance
- Multiple inheritance
- Multilevel inheritance

### 9. Method Overriding
- Redefining a parent class method in child class

### 10. Getters, Setters, and Deleters
- Used to control access to attributes
- Implemented using `@property`, `@setter`, `@deleter`

### 11. Composition vs Inheritance
- Inheritance: "is-a" relationship
- Composition: "has-a" relationship
